# Quiz App

A full-stack Quiz Application developed as part of an **internship project** using **React** for the frontend and **Node.js, Express, and MongoDB** for the backend. The project demonstrates core full-stack development concepts including user authentication, quiz management, RESTful APIs, and result processing in a clean and structured manner.

## Features
- User registration and authentication  
- Quiz category selection and question handling  
- Smooth quiz flow with automatic score calculation  
- Result display after quiz completion  
- Basic admin functionality for managing quiz questions  
- Responsive and user-friendly interface  

## Technology Stack
- Frontend: React, Axios, Bootstrap, Context API  
- Backend: Node.js, Express.js, MongoDB, Mongoose

## Local Setup
cd backend  
npm install  
npm start  

cd frontend  
npm install  
npm start  

Frontend runs on http://localhost:3000  
Backend runs on http://localhost:5000  

## Environment Variables
Create a `.env` file in the backend directory:  
MONGO_URI=your_mongodb_uri  
PORT=5000
